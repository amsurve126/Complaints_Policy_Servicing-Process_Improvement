# Insurance Complaints & Policy Servicing (BA Project)

This repository contains supporting artifacts for a business analysis project focused on improving the customer complaints and policy servicing process within an insurance organisation.

## Project Summary
- Analysed As-Is and To-Be complaint handling processes
- Conducted root cause analysis to identify drivers of repeat complaints
- Defined business and functional requirements
- Built an Excel-based operational dashboard for management reporting

## Key Artifacts
- Raw and cleaned complaint datasets
- Excel operations dashboard
- As-Is and To-Be BPMN diagrams
- Stakeholder analysis visuals
